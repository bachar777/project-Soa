# Client Management Frontend (SOA Project)

## Description du projet

Ce projet est une **application frontend en React** permettant la gestion des clients dans le cadre d’un projet.

L’application permet de :
- Afficher la liste des clients
- Rechercher un client par **nom** ou **ID**
- Ajouter un nouveau client
- Modifier les informations d’un client existant
- Supprimer un client
- Interagir avec une API REST backend via des requêtes HTTP

Les données sont récupérées depuis un service backend exposé via des endpoints REST.

---

## Technologies utilisées

- **React.js**
- **JavaScript (ES6+)**
- **Axios**
- **CSS**
- **Git & GitHub**
- **Backend REST API (SOA)**
